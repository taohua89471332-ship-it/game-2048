# 2048 Game Project

## Overview
2048 is a single-player puzzle game that combines elements of strategy and luck. The objective of the game is to slide numbered tiles on a grid to combine them and create a tile with the number 2048.

## How to Play
1. **Controls**: Use the arrow keys to move the tiles.
2. **Combining Tiles**: When two tiles with the same number touch, they merge into one.
3. **Goal**: Continue merging tiles until you reach the 2048 tile.

## Features
- Smooth animations and transitions.
- Ability to undo moves.
- Score tracking.
- Option to restart the game.

## Installation
To run the game locally:
1. Clone the repository: `git clone https://github.com/taohua89471332-ship-it/game-2048`
2. Open the `index.html` file in a web browser.
